Cybersecurity Workshop Outline

30 Minutes: Theory Session

Overview of social engineering tactics, malware analysis basics, and website security awareness.

65 Minutes: Practical Session

  • Social Engineering:
    • Using Zphisher for phishing simulations.
    • Demonstrating Grabify for IP tracking.
    • Exploring Iplogger for detailed IP logging.
  • Malware Analysis:
    • Understanding malware behavior and sandbox environments.
  • Hardware-based Wi-Fi Deauthentication:
    • Using an MCU node for executing Wi-Fi deauthentication.
  • Website Security Awareness:
    • Best practices for protecting websites from common vulnerabilities.

20 Minutes: Q&A Session

Open floor for questions and clarifications on the discussed topics and practical exercises.
